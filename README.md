# Project Title

A C++20 project.

## Requirements

- C++20 compatible compiler (e.g., GCC 10+, Clang 11+, MSVC 2019+)
- CMake (optional, for build management)
- Ninja

## Build Instructions

```sh
mkdir build && cd build
cmake -G Ninja .. -DCMAKE_BUILD_TYPE=Release
ninja
```
